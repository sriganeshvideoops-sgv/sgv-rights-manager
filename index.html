🔥 Firebase Sync

Sync your database using Google Firebase — free, real-time, trusted by Google. Your whole team shares one live database.

Setup — 3 minutes
  1. Go to console.firebase.google.com
  2. Click Add project → name it SGV Rights Manager → Create
  3. In left menu → Build → Realtime Database
  4. Click Create Database → choose location → Start in test mode → Enable
  5. Copy the Database URL (looks like https://sgv-xxx-default-rtdb.firebaseio.com)
  6. Paste it below → Connect
